Что такое Alba?

📡

Свой протокол RUTP

Reliable UDP Transport Protocol — разработан специально для Alba. Обеспечивает гарантированную доставку, контроль перегрузки (NewReno) и выборочные подтверждения SACK.

🔒

Безопасность и приватность

Сквозное шифрование, отсутствие метаданных на серверах. Только вы и ваш собеседник.

🐍

Клиент на Flet + Python

Единый код для Windows, macOS, Linux, iOS и Android. Быстро, красиво, родная производительность.

Почему Alba лучше?

🚀

Молниеносная скорость

UDP-база + умные алгоритмы дают минимальную задержку. Никаких Head‑of‑Line блокировок.

🔄

Адаптивное управление перегрузкой

Алгоритм NewReno (RFC 5681): медленный старт, избежание перегрузки, быстрая повторная передача.

💾

Умный буфер переупорядочивания

Принимаем пакеты в любом порядке, собираем поток заново — как в TCP, без потерь.

⏱️

Zero‑Window Probing

Persist-таймер и Keep‑Alive — соединение живёт, даже если у получателя закончилось место.

RUTP — надёжность UDP без компромиссов

RUTP (Reliable UDP Transport Protocol) — наша открытая реализация, написанная на Python с использованием asyncio. Протокол включает:

  • ✅ 32-битные циклические номера последовательностей (RFC 1982)
  • ✅ Трёхэтапное рукопожатие (SYN, SYN-ACK, ACK)
  • ✅ Окно перегрузки (cwnd) и медленный старт
  • ✅ Быструю повторную передачу после 3 дублированных ACK
  • ✅ SACK-блоки для выборочного подтверждения
  • ✅ Таймеры RTO с экспоненциальным отбоем
  • ✅ Контроль потока (Receive Window)
